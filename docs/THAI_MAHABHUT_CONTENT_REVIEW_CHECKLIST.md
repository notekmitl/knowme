# Thai Astrology — Content Review Checklist (Mahabhut)

Run this checklist for **every** candidate before moving it `reviewed →
canonApproved`. Items marked **(auto)** are pre-checked by the Review Assistant
in the Reviewer Workspace; items marked **(manual)** require human judgement and
cannot be automated.

Status: **CURRENT** · Decision Log **D-055**, **D-057** · paired with
`THAI_MAHABHUT_CANON_STYLE_GUIDE.md`.

> A candidate may only be approved when **all auto items pass** and **all manual
> items are confirmed by the reviewer**. The reasoning engine reads only
> Canon-Approved units.

---

## Per-unit checklist

- [ ] **Faithful structured knowledge (manual).** The `statement` records the
      book's concept/rule/logic faithfully — **not** a copied paragraph. No
      copyrighted narrative text is stored in the canon.
- [ ] **Book reference present (auto).** A `page` (+ chapter/section) locates the
      source. A verbatim quote is **not** required; provenance is by reference.
- [ ] **Page correct (auto + manual).** A `page` is set; the reviewer confirms it
      matches the printed page.
- [ ] **No added interpretation (manual).** Nothing is asserted that the book
      does not state. Empty fields stay empty rather than being filled by
      inference.
- [ ] **Metadata complete (auto).** `type`, `topic` and `subject` are set and
      follow the Style Guide naming.
- [ ] **Cross references complete (auto + manual).** Related units are linked;
      rules are not left orphaned; targets resolve.
- [ ] **Consistency (auto).** No duplicate-id, duplicate-formula or
      concept-naming conflicts (see Consistency tab).
- [ ] **Extraction notes recorded (manual).** Any book problem (duplicate text,
      archaic spelling, ambiguous table) is noted, with the source left
      unchanged.

## Per-chapter gate

Before a chapter is considered done, confirm in the Coverage tab:

- [ ] Chapter coverage and section coverage reflect all intended passages.
- [ ] Citation coverage is 100% (no unit without a page/book reference).
- [ ] Validation coverage is 100% (no outstanding validation errors).
- [ ] Consistency report is clean for the chapter.

---

## Workflow reminder

```
Candidate → (Validation Engine clean) → Validated
          → (this checklist confirmed)  → Reviewed
          → (approver sign-off)          → Canon Approved → Knowledge Index
```

Approval is the **only** step that admits content to the Canon Database. Nothing
on this list authorises creating knowledge that is not in the book.
